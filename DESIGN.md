# DESIGN SYSTEM - Estilo Dribbble-inspired

Resumo
- Estilo: moderno, gradientes suaves, cards com sombras suaves e bordas arredondadas.
- Tipografia: Poppins / Inter (títulos Poppins Bold, corpo Inter Regular)

Paleta
- Primary Gradient: linear-gradient(135deg, #6C5CE7 0%, #00B7FF 100%)
- Accent: #FF8A65
- Background: #071021
- Card Background: #0B1220
- Surface: #0E1A2B
- Text primary: #E6EEF8

Spacing
- base: 8px
- gap médio: 16px
- container padding: 24px

Componentes iniciais
- Header (logo, perfil)
- Dashboard (cards de curso / progresso)
- StudyCard (título, tempo, progresso)
- Botões primário / secundário

Assets
- /assets/design/mockup-hero.txt (placeholder para mockup PNG)

Como usar
- Esta branch contém exemplos React com styled-components. Para testar localmente, instale dependências e importe as fontes Poppins e Inter no index.html ou via @import no CSS.
